The Best and the Brightest

The Best and the Brightest is a is a proof of concept for a narrative game optimized for streaming. The streamer plays as the president of the fictional Federation of Sovereignty making decisions over how to run the country, Reigns style. Selected subscribers provide information to the president as secretaries in the president's cabinet. The president has ninety seconds to decide on yes or no question amid the chaotic discussion of the secretaries and the chat.

This project was done as a group project(Myself, Nick Banducci and Zac Emerzian). This project was done as a requirement for the course titled GAME 270.

Tools used: Unity Editor, C#
