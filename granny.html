Grandma's Last Hope

Grandma's Last Hope is a 2D platformer game. The game is about a Grandma who has to save her grandson from bullies by just using a single item "Grandma's hand made Scarf". She only has one life and the game is only one level. The game concept is inspired from Indiana Jones and Mama in Hobbs and Shaw.

This project was done as a group project(Myself and Nancy Lin). This project was done as a submission to the GMTK Game Jam 2019 which lasted for 48 hours.

Tools used: Unity Editor, C#



I worked as the game programmer and sound engineer for this project. I implemented the rope mechanics and the Granny's and bullies movements.
