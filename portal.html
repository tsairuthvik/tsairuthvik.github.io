AR Portal

Augmented Reality Portal

This was a solo project. This video shows you about a project I have been working on for a while. We can use any 360 video in this project and the portal immerses you into the video.

I have used ARCore for this project as it works efficiently on (select) Android phones. Hopefully, Google expands the ARCore availability to more phones!

This is just a minor prototype of a large project that I am currently working on, so stay tuned to more videos like this!


Tools used: Unity3D, ARCore, C#, Pixel 2XL
