Multiplayer Ninja Race

Multiplayer Ninja Race is a proof of concept for a 2d platformer race game. A 2d platformer race where you have to reach the finish line(duh..) but in the process you can kill your opponent to slow him down, the project is in the development stages and this is just a proof of concept of multiplayer using PUN 2 and Unity.

This project was done so that I can understand the basic concepts of a multiplayer system, how the matchmaking works and how the player interacts with the environment and how they see the other player interacting with the environment, the leaderboard system etc.

Tools used: Unity Editor, C#, PUN 2
