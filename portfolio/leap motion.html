Augmented Reality Game Object Control using Leap Motion

A state of the art implementation of an Augmented Reality Game Object Control using Leap Motion

I worked on this project as a research assistant at the Cognitive Neuroscience lab, BITS Pilani. I worked on integrating three concepts in this project namely BCI, AR and Home Automation. I developed the AR app for both Android and iOS devices, which received the Raw EEG data from the EEG headset. Blink Strength, Attention and Medtation levels are calcualted in the App in Real time and are used as thresholds for switching on/off a particular device. I also worked on the communication(WiFi) between the AR app and the Arduino associated with the device. Device used was a Table Light.

The project can be upgraded to controlling an electric wheelchair also

Tools used: Unity Editor, C#, Leap Motion
