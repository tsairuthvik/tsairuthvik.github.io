Integration of Leap Motion with AR

A state of the art implementation of Augmented Reality version of Pinch Drawing effect using Leap Motion

I created this project with the idea of AR Painting using Pinch Drawing effect of the Leap Motion. Marker based AR system was used for this project for the AR experience. Hence, a bluestar Logo image was used as the image target and the Leap Motion for the hand tracking. I also added the hand models to the scene, hence when the hands are tracked, the virtual hand models are displayed on the image target(bluestar Logo). The future uses of this technology can be an Augmented Reality classroom where the teacher can use this for teaching instead of a Chalk on a blackboard, or the students can take notes on a specific image target and can later view it.

Tools used: Unity Editor, C#, Leap Motion
