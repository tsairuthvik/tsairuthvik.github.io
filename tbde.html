The Best Date Ever

The Best Date Ever is a proof of concept for a narrative game that brings out a specific emotion out of the player. The player plays as a person on a date with a beautiful girl and the game is showcases specific moments during the date.

This project was done as a group project(Myself, Yehe Lin and Yupeng Li). This project was done as a requirement for the course titled GAME 270.

Tools used: Unity Editor, C#
